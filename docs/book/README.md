# shrike-book

Reading path for understanding shrike, the decisions behind it,
and how to use it effectively. Short chapters by design.

1. [01-intro.md](01-intro.md) — ROP in 2026, tool landscape,
   where shrike fits.
2. [02-taxonomy.md](02-taxonomy.md) — gadget classification,
   canonical dedup, what each category is for.
3. [03-chains.md](03-chains.md) — recipe DSL, multi-pop
   synthesis, clobber graph, auto-padding.
4. [04-verification.md](04-verification.md) — SMT-LIB2 proof
   emitter, Z3 flow, what it catches.
5. Loaders and Integration chapters are stubbed — will grow
   when a real use case shows up that the existing docs don't
   cover.

Start with 01 if you're new. Skim the TOCs if you already
know ROP — chapters are written to be read out of order.
